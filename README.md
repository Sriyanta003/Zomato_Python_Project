# Zomato_Python_Project
Zomato Data Analysis Project using Python. Performed data cleaning, exploratory data analysis (EDA), and visualizations using Pandas, Matplotlib, and Seaborn to analyze restaurant types, ratings, votes, and online ordering trends.

🍽️ Zomato Data Analysis Project

📌 Overview

This project performs Exploratory Data Analysis (EDA) on a Zomato restaurant dataset using Python. The analysis focuses on understanding restaurant trends such as ratings, votes, online ordering patterns, and cost distribution using data visualization techniques.

📂 Dataset

The dataset contains information about restaurants such as:

Restaurant Name
Online Order Availability
Table Booking
Ratings
Number of Votes
Approximate Cost for Two People
Restaurant Category (Buffet, Cafes, Dining, etc.)

🛠️ Tools & Technologies

🐍 Python
📊 Pandas
🔢 NumPy
📈 Matplotlib
🎨 Seaborn
📓 Jupyter Notebook

🔍 Project Workflow

1️⃣ Data Loading using Pandas
2️⃣ Data Cleaning and Preprocessing
3️⃣ Handling and converting rating values
4️⃣ Exploratory Data Analysis (EDA)
5️⃣ Data Visualization using Matplotlib & Seaborn

📊 Visualizations

This project includes several visualizations:
📌 Count plot of restaurant categories
📌 Line chart showing votes by restaurant type
📌 Histogram of ratings distribution
📌 Cost distribution analysis
📌 Boxplot comparing online vs offline order ratings
📌 Heatmap showing relationship between restaurant type and online orders

📈 Key Insights

🍴 Most restaurants belong to the Dining category.
⭐ Dining restaurants receive the highest number of votes.
📊 Most restaurant ratings fall between 3.5 – 4.0.
💰 Many couples prefer restaurants costing around ₹300 for two people.
📱 Restaurants offering online ordering often receive higher ratings.
